Назначение модуля:
- Интеграция с системой учета заявок ТО Библио Глобус
- Контроль статусов заявок
- Контроль корректности информации

Функции:
- Страница администрирования (формы конфигурации)
  - Настройка учетной записи
  - Привязка сущностей к заявкам в системе БГ:
    - Какие бандлы каких типов сущностей являются заявками
    - Какое поле является указателем на номер заявки
    - [OPTIONAL] Какое значение другого поля указывает на БГ
- Запрос информации о заявках из системы БГ
  - Организация HTTP-запросов и кэширования
- Поля, показывающие статус заявки, статус оплаты
  - Форматтеры
  - extra fields
  - Интеграция с views
